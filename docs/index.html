2 - To set as Default Homepage

To set Homepage+ as your default homepage, there are ways for both Firefox Users and Google Chrome users on this documentations page. For Internet Explorer, Microsoft Edge or Opera, please refer to Google (or your preferred search engine).


2.1 - Firefox

To set Homepage+ as your Homepage on Firefox:

First, click on the menu tab on the top-right of your firefox screen. In the drop-down menu, click "Options".

After going into the preferences page, on the left side there will be multiple options. Click on Home.

Under New Windows and Tabs, in the drop down menu for Homepage and new windows, select "Custom URLs..."

Under the textbox for "Custom URLs...", and enter this link (for Homepage+):

http://homepage.huzz.xyz/

And you're done! Next time you start up your Firefox, your homepage should load Homepage+! Keep reading for how to set your New Tab page to be Homepage+.
